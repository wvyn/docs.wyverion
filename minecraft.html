Wyverion 1.12.2 Modpack

Install Forge 1.12.2

To run this modpack, you need Forge 1.12.2 installed. Download the installer below:

Download Forge 1.12.2 Installer

How to install:

  • Run the forge-1.12.2-14.23.5.2860-installer.jar
  • Select "Install Client" and click OK
  • Open the Minecraft Launcher and choose the Forge 1.12.2 profile

How to Install Mods

Once you’ve installed Forge and downloaded the modpack:

  • Open your Minecraft folder:
    • Windows: Press Win + R, type %appdata%\.minecraft, and press Enter
    • Mac: Go to ~/Library/Application Support/minecraft
  • Create a mods folder if it doesn’t exist
  • Extract the contents of halfNewMods1.zip/halfNewMods2.zip into the mods folder
  • Start Minecraft with the Forge 1.12.2 profile

How to Join the Server

  • Launch Minecraft with the installed mods
  • Click Multiplayer
